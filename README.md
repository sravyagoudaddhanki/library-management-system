# Library Management System

## Project Overview

The Library Management System is a simple Python-based application
that helps manage books in a library. The system allows users to add
books, view available books, search books, issue books, return books,
and maintain library records automatically. The application behaves
like a basic library assistant by managing book information and user
interactions through the terminal.

## Project Objective

Develop a Python Library Management System that can:

- Add new books
- Display available books
- Search books
- Issue books to users
- Return books
- Maintain library records
- Save data automatically

## Project Description

The application works in the terminal/command prompt and allows
librarians to manage books efficiently. The system stores book
details, allows adding new books, displays all books available in
the library, searches books by title or author, issues books to
users, accepts returned books, and updates library records
automatically.
